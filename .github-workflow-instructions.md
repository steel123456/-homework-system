# 部署步骤指南

## 第一步：创建 GitHub 仓库

### 方式 A：网页创建（推荐）
1. 打开 https://github.com/new
2. 填写信息：
   - Repository name: `homework-system`
   - Description: `智能作业管理系统 - 班级管理、作业布置、AI批改`
   - 选择 **Private**（私有）或 **Public**（公开）
   - ❌ 不要勾选 "Add a README file"
   - ❌ 不要勾选 "Add .gitignore"
   - ❌ 不要选择 License
3. 点击 **Create repository**

### 方式 B：命令行创建（需要 GitHub CLI）
```bash
gh repo create homework-system --private --source=. --remote=origin
```

## 第二步：推送代码到 GitHub

创建仓库后，在项目目录执行以下命令：

```bash
# 添加远程仓库
git remote add origin https://github.com/steel123456/homework-system.git

# 推送代码
git push -u origin main
```

如果提示登录，输入你的 GitHub 用户名和 Personal Access Token（不是密码）。

### 如何获取 Personal Access Token？
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 点击 "Generate token"
5. 复制生成的 token（只显示一次）

## 第三步：部署到 Vercel

1. 访问 https://vercel.com
2. 点击 "Sign Up" → 选择 "Continue with GitHub"
3. 授权 Vercel 访问你的 GitHub
4. 点击 "Add New..." → "Project"
5. 选择 `homework-system` 仓库
6. 点击 "Import"

### 配置环境变量

在 Vercel 的 "Environment Variables" 中添加：

```
NEXT_PUBLIC_SUPABASE_URL=你的supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的supabase_anon_key
```

（这些值需要你先在 Supabase 创建项目后获取）

7. 点击 "Deploy"
8. 等待约 2 分钟完成部署

## 完成！

部署后你会获得：
- 永久访问地址：`https://homework-system.vercel.app`
- 每次推送代码，Vercel 会自动更新
